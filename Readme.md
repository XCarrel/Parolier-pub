# Parolier

Nous sommes nombreux à déplorer le fait que Qobuz ne permette pas d'afficher les paroles des chansons qui sont jouées.  

Cette application propose de combler (momentanément peut-être) ce manque. Elle propose une solution fonctionnelle bien que nettement plus compliquée que ce que l'on souhaiterait. Mais bon, en attendant ...

Mise en place :
- Se créer (si nécessaire) un compte sur [last.fm](https://www.last.fm)
- Se créer une API Application [ici](https://www.last.fm/api/account/create) sur last.fm
- Activer le "scrobbling": sur le compte personnel [qobuz](https://www.qobuz.com), aller dans "mes services externes" pour connecter les comptes qobuz et lastfm  
- Installer et démarrer Parolier sur un [Mac récent](https://github.com/XCarrel/Parolier-pub/releases/download/v1.0/Parolier_Mac_Apple_Silicon-1.0.0.dmg), un [Mac plus ancien](https://github.com/XCarrel/Parolier-pub/releases/download/v1.0/Parolier_Mac_Intel-1.0.0.dmg) (puce Intel) ou un [PC](https://github.com/XCarrel/Parolier-pub/releases/download/v1.0/Parolier.Setup.1.0.0.exe)
- Introduire les informations de l'API dans "Configurer last.fm"

Enjoy!
